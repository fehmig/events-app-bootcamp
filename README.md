## Özet
React Native ile geliştirdiğim uygulamada çeşitli kategorilere ait etkinlikler görüntülenmesi ve bu etkinliklere ait detaylara ulaşılması amaçlanmıştır. Etkinliklere ait detaylar kullanıcı deneyimini üst seviyeye çıkarmak isteğiyle tasarlanmıştır.  Favorilere ekleme, sepete ekleme, satın alım deneyimi (UI) gibi işlevler yapılabilmektedir. State yönetimi Zustand ile sağlanmıştır. Lottie, IcoMoon kullanılarak göze hoş gelen tasarım üzerinde çalışılmıştır. Ayrıca lokasyon bilgileri Google Maps API üzerinden detay sayfalarında map üzerinde gösterilmiştir.
